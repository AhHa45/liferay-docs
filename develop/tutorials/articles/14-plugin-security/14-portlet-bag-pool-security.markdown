# Portlet Bag Pool Security

[TOC levels=1-4]

Specify regular expression patterns used to match any portlet IDs that the
plugin is permitted to access from the portlet bag pool. 

*Example:*

    security-manager-portlet-bag-pool-portlet-ids=\
        1_WAR_flashportlet,\
        .*_WAR_chatportlet
